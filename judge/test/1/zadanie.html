Máme dve rôzne zaklinadla - chceme vedieť, ktoré je silnejšie. Všetci vieme že lexikograficky menšie zaklínadlá sú silnejšie. Ale pozor, lebo v magickom jazyku existuje iné abecedné poradie...

Input:

INT dumbier: dĺžka prvého zaklínadla
STR recept: prvé zaklínadlo (dumbier lowercase znakov)
INT mandragora: dĺžka druhého zaklínadla
STR zaklinadlo: druhe zaklínadlo (mandragora lowercase znakov)
STR zaba: permutácia anglickej abecedy - magické lexikografické usporiadanie pismen
INT jedna = 0: pomocná premenná
INT netopier = 0: pomocná premenná

Output:

STR zaklinadlo: lexikograficky menšie zaklínadlo určeného žabou.
