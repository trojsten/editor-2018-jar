Vojaci idu cez obdlznikove bojovisko. Na zaciatku je kazdy modry. Ked niekto stretne ‘p’, to je priest, a je modry, zmeni sa na cerveneho. Ked stretne ‘P’, to je modry priest, a je cerveny, zmeni sa na modreho. Spocitajte, kolko krat vojaci zmenia farbu.

V kazdom riadku bojoviska je vojak, a ide zlava doprava

input:
INT N: velkost bojoviska (N*N matrix)
VECTOR<STRING> bojovisko: N stringov dlzky N, pozostavajucich z pismen
INT wololo: premenna na odpoved
INT modry: pomocna premenna
INT cerveny: pomocna premenna
INT modrocerveny: pomocna premenna
INT priest

output:
INT wololo: kolko krat vojaci zmenil farbu
