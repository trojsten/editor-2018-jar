Uplne dominujeme mapku, ale este sme nevyhrali. Vyhrame o N sekund. Prave sme dobyli nejaku zlatu banu, spawnol sa pri nej nespokojny magicky domorodec ktory zautoci na nasich workerov. Chceme tomu zabranit. Mame na vyber M <= 10^4 typov bojovnikov, i-ty z nich ma H_i zivotov a stoji C_i penazi (H_i, C_i <= 1000). Kedze uplne dominujeme, mame k dispozicii 10^6 <= Z <= 10^9 zlata.

Nespokojny magicky domorodec ma kuzlo, ktore robi dokopy D_i damage okolostojacim vojakom. Kedze je magicky, nasi vojaci mu nerobia damage, ale len smutne stoja a tankuju ho.

Nech sum(D) je sucet D_1,...,D_{i-1}, a len(x) pocet cislic v cisle x
D_i = 0, ak i = 0,
D_i = ( D_{i-1} * magia+sum(D)%i + 47 + len(D_{i-1}) ) % 1000 ak i>0 a D_{i-1} je neparne,
D_i = ( D_{i-1} * len(sum(D)) + i%47 + magia*31 + sum(D)%len(magia) ) % 1000 ak i>0 a D_{i-1} je parne
(a my musime vydrzat sekundy 1...N)

Kolko najmenej vojakov potrebujeme, aby prezili N sekund?

input:
INT N: pocet sekund kym vyhrame. 1<=N<=100
INT Z: pocet zlata ktore mame. 10^7 <=Z<= 10^9
INT M: pocet typov bojovnikov na vyber. 1<=M<=10^4
VECTOR<INT> bojovnici: M cisel, reprezentujucich zivoty roznych typov bojovnikov 1<=zivoty<=10^4
VECTOR<INT> nakupneceny: M cisel, reprezentujich ceny roznych typov bojovnikov. 1<=ceny<=100
INT magia: Magicke cislo magickeho domorodca. 1<=magia<=1000

output:
INT magia: najnizsi pocet bojovnikov ktorych potrebujeme, aby vydrzali aspon N kuziel.
